# Call To Action Component

The `Call To Action` component takes a `cta` object and renders it as a `Button`. (See `Button` [molecule](?path=/docs/molecules-button--docs) for more details).

<Canvas of={CallToActionStories.Default} />

<br />

## cta

The `Call To Action` component accepts a `cta` object with the following properties, which are set in PayloadCMS:

```
{
    color?: string;
    label?: string;
    linkType?: string;
    externalLink?: string;
    internalLink?: object;
}
```

### color

The button's color.

<hr />

### label

The button's label.

<hr />

### linkType

The button's link type; determines which link is utilized.

<hr />

### externalLink

The button's external link (used if `linkType` is `external`).

<hr />

### internalLink

The button's internal link (used if `linkType` is `internal`).

<br />
