## buttonData

### PayloadCMS

From dashboard, navigate to `Content` > `Buttons`.

Buttons contain the following fields (these vary depending on which `Link Type` is selected):

-   `Label` - input button's label
-   `Link Type` - select whether button link is internal or external
-   `Internal Link` - select site page to link to
-   `External Link` - input valid external URL
-   `Color` - select button's color
