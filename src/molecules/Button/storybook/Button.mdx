# Button Molecule

<Canvas of={ButtonStories.Default} />

<br />

## children

Text is supplied as a child to the `Button` molecule.

## color

Controls the color of the `Button` molecule.

## link

If the link prop is not empty, the `Button` will render as an `a` tag. Otherwise, it will default to a `button` tag.

## onClick

If there is no link and the `onClick` prop is supplied, it will be called when the `Button` is clicked.
