# Icon Atom

<Canvas of={IconStories.Default} />

<br />

### id

The icon `id` is used to select which icon to display.

<hr />

### size

Select a size for the `Icon` atom. If no size is selected, size will default to `sm`.

<hr />

### color

Select a color for the `Icon` atom's fill. If no color is selected, color will default to `gray`.

<hr />

### ref

An optional ref that is forwarded to the underlying SVG element.
