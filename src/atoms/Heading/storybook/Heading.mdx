## children

Text is supplied as a child to the `Heading` atom.

## as

Select a semantic heading tag using best practices to keep the page accessible. Read this article from [The A11y Project]('https://www.a11yproject.com/posts/how-to-accessible-heading-structure/#best-practices-summarized') for more information on best practices.

If the `as` prop is not defined, the tag will default to `h2`.

## color

Select a color for the `Heading` atom. If no color is selected, color will default to `green`.

## size

Select a size for the `Heading` atom. If no size is selected, size will default to `md`.
