# Button Atom

<Canvas of={ButtonStories.Default} />

<br />

### `children`

Text is supplied as a child to the `Button` atom.

### `color`

The color of the `Button` is controlled by the `color` prop.

### `link`

If the link prop is not empty, the `Button` will render as an `a` tag. Otherwise it will default to a `button` tag.

### `onClick`

If the `onClick` prop is supplied, it will be called when the `Button` is clicked.
