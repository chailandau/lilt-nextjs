# Image Atom

<Canvas of={ImageStories.Default} />

## src

Supply a path to an image compatible with the Next.js built-in `Image` component. Read more about it [here](https://nextjs.org/docs/pages/api-reference/components/image).

## alt

Supply alt text for the image. This is required, as alt text is important for accessibility. Read more about alt text best practices on [WebAIM](https://webaim.org/techniques/alttext/).

## priority

If true, lazy loading is disabled and the image is preloaded.

## hasBorder

If true, image will be rounded and have a double border.
