# Link Atom

The `Link` atom wraps the `next/link` component. There are lots of goodies built in, like prefetching. Read more about it [here](https://nextjs.org/docs/pages/api-reference/components/link).

<Canvas of={LinkStories.Default} />

<br />

### children

Text is supplied as a child to the `Link` atom.

<hr />

### href

The link destination.

<hr />

### className

Add an optional CSS class name.

<hr />

### underline

Adds an underline effect on hover. Defaults to `true`.
